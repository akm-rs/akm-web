# Project LLM Instructions

akm-web is the website for the Agnet Kit Manager CLI tool - akm
akm is available at github.com/akm-rs/akm

akm's purpose is to offer automated and centralised management of the LLM agent kit : 
- skills
- artifacts
- instructions

# Website

This website is hosted with github pages and available at akm.raphaelsimon.fr for now

# Tech stack

- Jekyll 
- Ruby for live preview

# Editing the website

Any and all changes must happen through a PR and be live previewed for test before merge.
GH Pages deploys from main !
